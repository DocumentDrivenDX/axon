### Verdict: REQUEST_CHANGES

### Summary

The plan closes many obvious holes, but it still leaves transaction-level CDC behavior, snapshot consistency, migration exclusivity, legacy recovery reads, canonical payload measurement, and delete/link evolution semantics underdefined. Those gaps can produce incompatible implementations or false readiness even if every listed phase is executed. The required changes are mostly contract clarifications plus targeted failure and parity tests before implementation beads proceed.
