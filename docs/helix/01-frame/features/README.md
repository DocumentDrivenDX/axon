# Feature Specifications

**Project**: Axon
**Last Updated**: 2026-06-10

This directory holds one specification per feature. The authoritative catalog
of feature IDs, names, priorities, statuses, dependencies, and trace links is
the [Feature Registry](../feature-registry.md) — the table formerly kept here
has moved there and is no longer maintained in this file.

## Conventions

- One file per feature: `FEAT-XXX-short-name.md`
- Feature IDs are assigned sequentially and never reused
- All features trace back to PRD requirements
