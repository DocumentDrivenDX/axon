# Sample Projects

The sample suite is generated with the coverage catalog for Axon 0.7.1.
Each project includes schemas, seed data, typed links, and a demo script that can
be adapted to a local Axon binary.

## Agent Taskboard

A governed task and bead queue that demonstrates collections, schemas, entity CRUD, links, graph traversal, audit, optimistic concurrency, MCP-oriented task discovery, and the unified CLI.

- Persona: Developers building agent-native workflow state.
- Source: [`examples/agent-taskboard`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/agent-taskboard)
- Demo reel: [agent-taskboard-reel](../demo-reels/agent-taskboard-reel/)
- Coverage entries: 33

## Invoice Approval Guardrails

A finance workflow covering AP/AR partial payment, state machines, policy envelopes, mutation intent preview/approval, rollback, and audit-safe trusted agent writes.

- Persona: Finance operators and trusted finance agents.
- Source: [`examples/invoice-approval-guardrails`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/invoice-approval-guardrails)
- Demo reel: [finance-guardrails-reel](../demo-reels/finance-guardrails-reel/)
- Coverage entries: 39

## Customer Identity Graph

CRM, CDP, and MDM flows for contact merge, identity resolution, golden record survivorship, relationship traversal, and provenance.

- Persona: Customer data teams and revenue operations.
- Source: [`examples/customer-identity-graph`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/customer-identity-graph)
- Demo reel: [customer-identity-reel](../demo-reels/customer-identity-reel/)
- Coverage entries: 11

## Supply Chain BOM

ERP-style bill-of-materials traversal, recursive graph queries, reachability checks, aggregation, and link metadata.

- Persona: Operations teams modeling product dependencies.
- Source: [`examples/supply-chain-bom`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/supply-chain-bom)
- Demo reel: [supply-chain-reel](../demo-reels/supply-chain-reel/)
- Coverage entries: 19

## Tenant Control Plane

Multi-tenant path routing, users, credentials, JWT grant bounds, revocation, deployment registration, and BYOC isolation.

- Persona: Operators running Axon for multiple tenants or deployments.
- Source: [`examples/tenant-control-plane`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/tenant-control-plane)
- Demo reel: [tenant-control-reel](../demo-reels/tenant-control-reel/)
- Coverage entries: 26

## Schema Release Sync

Schema evolution, secondary indexes, validation rules, CDC, markdown templates, git mirror output, and rollback preview.

- Persona: Developers shipping schema-backed applications.
- Source: [`examples/schema-release-sync`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/schema-release-sync)
- Demo reel: [schema-release-reel](../demo-reels/schema-release-reel/)
- Coverage entries: 42

## Admin Policy Workbench

Policy authoring, dry-runs, redacted browsing, intent queues, approval review, stale-intent handling, and UI parity with GraphQL/MCP.

- Persona: Administrators and reviewers using the Axon web UI.
- Source: [`examples/admin-policy-workbench`](https://github.com/DocumentDrivenDX/axon/tree/master/examples/admin-policy-workbench)
- Demo reel: [admin-policy-reel](../demo-reels/admin-policy-reel/)
- Coverage entries: 28
